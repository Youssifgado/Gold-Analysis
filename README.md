This SQL script analyzes sales data by joining fact and products tables. It includes:
  Basic joins to combine data.
  Aggregations to calculate total sales by product, date, and year.
  Use of window functions (AVG, LAG) for advanced metrics like average sales, yearly trends, and comparisons.
  Conditional logic (CASE) to classify sales performance and yearly progress.
  Common Table Expressions (CTEs) for reusable subqueries.
